## SKD Bundle Block Plugin

В репозиторий добавлен плагин WordPress `SKD Bundle Block`, который добавляет Gutenberg-блок «Комплект товаров» с тремя слотами для товаров.

### Возможности

- Настройка размера скидки и привязки категорий товаров к каждому из трех слотов через страницу настроек «Комплект товаров» в админ-панели WordPress.
- Вывод блока с автоматическим заполнением товаров из выбранных категорий (используется последний опубликованный товар).
- Резервное отображение подсказок, если WooCommerce недоступен или категории не заданы.

### Установка

1. Скопируйте содержимое каталога `wp-content/plugins/skd-bundle-block` в директорию `wp-content/plugins/` вашего проекта WordPress.
2. Активируйте плагин в админ-панели WordPress.
3. Настройте скидку и категории слотов на странице «Комплект товаров».
4. Добавьте блок «Комплект товаров» на нужную страницу/запись в редакторе Gutenberg.
